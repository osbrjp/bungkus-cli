# osbr-devtools artifacts

Portable Claude Code assets that belong in the **osbr devtools** company toolkit,
kept here in source so they stay versioned next to the CLI whose behavior they
describe (see [`../../CLAUDE.md`](../../CLAUDE.md) — update these when CLI flags change).

They are intentionally **not** under `.claude/skills/`, so they are not loaded as
live skills in this repo — copy them into osbr devtools to install.

- `scaffold-advisor/SKILL.md` — turns a plain-English brief into a `bungkus-cli create` command.
